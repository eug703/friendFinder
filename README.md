# friendFinder

### FriendFinder is an app that matches users with an ideal friend

*Deployed here:
https://friendfinderhwgw19.herokuapp.com

### Technologies Used
* Javascript
* JQuery
* Node.js
* Express

### Overview
* app
    * data
        * friends.js
    * public
        * home.html: Landing page
        * survey.html: Survey page
    * routes
        * apiRoutes.js
        * htmlRoutes.js
* node_modules
* package-lock.json
* package.json
* server.js

### App Instructions
* Clone from github 
* Command Line:
* npm install (Installs all dependencies)
* node server.js
